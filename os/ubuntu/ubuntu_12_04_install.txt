打开休眠功能：
需要用文本编辑器创建 
sudo vim /etc/polkit-1/localauthority/50-local.d/com.ubuntu.enable-hibernate.pkla 文件，并填写以下内容保存：


[Re-enable hibernate by default]
Identity=unix-user:*
Action=org.freedesktop.upower.hibernate
ResultActive=yes

注销后即可出现休眠选项，但是仍然会花屏。下面会安装显卡驱动，就能解决这个困扰我很久的问题。

lspci查看显卡型号为：
01:00.0 VGA compatible controller: Advanced Micro Devices, Inc. [AMD/ATI] RV710/M92 [Mobility Radeon HD 4530/4570/545v]

在AMD的官方网站http://www.amd.com/zh-cn找到合适的驱动并下载：
wget http://www2.ati.com/drivers/legacy/amd-driver-installer-catalyst-13.1-legacy-linux-x86.x86_64.zip
unzip amd-driver-installer-catalyst-13.1-legacy-linux-x86.x86_64.zip && chmod +x ./amd-driver-installer-catalyst-13.1-legacy-linux-x86.x86_64.run && ./amd-driver-installer-catalyst-13.1-legacy-linux-x86.x86_64.run && 选择自动安装 && reboot
